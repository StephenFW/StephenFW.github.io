Description

The <ul>, <ol>, and <dl> tags are used to define lists, the <ul> tag is used for unordered lists the <ol> tag is used for ordered lists and the <dl> tag is used for description lists. The <li> tag is used to define a list item and in a description list the <dt> tag defines a list item and the <dd> is used to describe a list item.

Syntax

<ul, ol, or dl>

<li or dt>Text</li or /dt>

<li or dt>Text</li or /dt>

</ul, /ol, or /dl>

Example

<ul>
<li>Alpha</li>
<li>Beta
<ul>
<li>One</li>
<li>Two</li>
</ul>
</li>
<li>Alpha</li>
</ul>

Display

List

  • Alpha
  • Beta
    • One
    • Two
  • Gamma
