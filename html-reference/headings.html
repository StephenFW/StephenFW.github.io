Headings

Description

The heading tags are used to mark text as a heading, you can number the heading to desplay the text as being somewhat smaller to denote a "lower" heading.

Syntax

<h1>Heading</h1>

You can also use h2, h3, h4 ... etc.

Example

<h1>This is a heading</h1>
<h2>This is a heading</h2>
<h3>This is a heading</h3>

Display

This is a heading
