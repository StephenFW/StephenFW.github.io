Blocks

Description

The <div> tag is used to sort of "divide" the <body> into different sections so that you can apply a style into each section and it will not affect the other sections.

Syntax

<body>

<div>

HTML

</div>

</body>

Example

<body>

<div>

One

</div>

<div>

Two

</div>

</body>

Display

One
Two
