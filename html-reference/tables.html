Tables

Description

The <table> tag is used to define a table, the <tr> tag is used to define a row on a table and the <th or td> tag is used to define boxes in each row in order, <td> is used for the top row or the "title" and <th> is used for the other rows.

Syntax

<table>

<tr>

<th>Text</th>

<th>Text</th>

</tr>

<tr>

<td>Text</td>

<td>Text</td>

</tr>

</table>

Example

<html>
<head>
<style>
table, th, td {
border: 1px solid black;
border-collapse: collapse;
}
th, td {
padding: 5px;
text-align: left;
}
</style>
</head>
<body>

<h2>Table</h2>
<table style="width:100%">
<tr>
<th>One</th>
<th>Two</th>
<th>Three</th>
</tr>
<tr>
<td>Uno</td>
<td>Dos</td>
<td>Tres</td>
</tr>
</table>

</body>
</html>

Display

Table

One Two Three
Uno Dos Tres
