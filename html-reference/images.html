Images

Description

The <img> tag display the image denoted as a link, if the image doesn't load, the "alt text" is what will display.

Syntax

<img src= "[link]" alt= "[text]">

Example

<img src= "https://www.google.com/images/branding/googlelogo/2x/googlelogo_color_272x92dp.png" alt= "Google">
