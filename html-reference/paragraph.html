Paragraphs

Description

The <p> tag is used to define a block of text as a paragraph, this is different from just typing the text because you can split the text into groups that have spaces.

Syntax

<p>Paragraph text</p>

Example

<p>This is a paragraph</p>
<p>This is a paragraph</p>
<p>This is a paragraph</p>

Display

This is a paragraph

This is a paragraph
