HTML

Description

The <html> element is used to define the part of a document that contains HTML. Basically a tag used to contain all HTML code.

Syntax

<html> text </html>

Example

<!DOCTYPE html>

<html>

<head>

</head>

<body>

<p>This is a paragraph</p>

</body>

</html>

Display

This is a paragraph

Title

Description

The <title> element will display the text within it in the taskbar of your computer browser. Basically sets the tab name of a specific page in your website.

Syntax

<title> text </title>

Example

<title> This is a title </title>

Display

*The title of the tab should be called "This is a title"

Headings

Description

The heading element is used to display the text within it as a heading, the higher the number that you set the heading tag to (h1, h2, h3 etc.) the smaller the heading becomes.

Syntax

<h1> text </h1>, <h2> text </h2>, <h3> text </h3> etc.

Example

<h1> This is a heading </h1> <h2> This is a heading </h2> <h3> This is a heading </h3>

Display

This is a heading

This is a heading

This is a heading

Paragraphs

Description

The paragraph element will define a block of text as a paragraph, the paragraph would not be indented and two different paragraphs next to eachother will be separated from eachother by a margin and by a break instead of merging.

Syntax

<p> text </p>

Example

<p> This is a paragraph </p>

Display

This is a paragraph

Breaks

Description

The break element acts as the "enter" key on your keyboard, when inserted in a block of text it would move the text down to the next line. Making two separate paragraphs out of one block of text will do something similar.

Syntax

<br>

Example

This is some text...
<br>
...and this is some more text...

Display

This is some text...
...and this is some more text...

Lines

Description

The line element will create a horizontal line wherever you insert it, usually you would want to put HTML code above and below the line tag and not on the same row.

Syntax

<hr>

Example

This is some text...
<hr>
...and this is some more text...

Display

This is some text...
...and this is some more text...

Comments

Description

The comment element will mark a block of text so that it would not display as a result of running your code but can be seen when viewing the code itself, comments can be used to write private notes as well as organize your code.

Syntax

<!-- text -->

Example

<!-- This is a comment -->

Display

*The text will not display as a result of running the HTML

Links

Description

The link element is used to display a link in your webpage, you can set the display text of the link and of course the link itself. Links are colored blue and underlined by default but can be formatted.

Syntax

<a href= "[link]"> text </a>

Example

<a href= "https://goo.gl/o6zvxV"> This is a link </a>

Display

This is a link

Images

Description

The image element is used to display an image as long as you have the image link, if the image fails to load you can add "alt" text to display when the image doesn't load, informing a possible viewer that the image is not working.

Syntax

<img src= "[url]" alt= "text">

Example

<img src= "https://goo.gl/o6zvxV" alt= "Alt text">

Display

Lists

Description

The list element is used to create a list. A list could be an ordered list, an unordered list, and a description list. You can also set sub-items and use either a numbered list format, dots, dashes, and even squares.

Syntax

<ul, ol, or dl>

<li or dl> text </li or /dl>

</ul, /ol, or /dl>

Example

<ul>

<li> This is some text </li>

<li> This is some more text </li>

<li> This is some more text </li>

</ul>

Display

  • This is some text
  • This is some more text
  • This is some more text

Blocks

Description

The block element or the "div" element is used to split the "body" of a document into multiple sections. Within these sections you can put seperate lines of code, a style applied to a section will not be applied to other sections.

Syntax

<div> text </div>

Example

<div style="color:red">
<p>This is some text</p>
</div>

<p>This is some more text</p>
